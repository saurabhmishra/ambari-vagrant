ambari-vagrant
==============
Modified Vagrant scripts for Ambari to spin up 1 Node cluster by running below commands.

Ambari Version: 1.5.1
HDP 2.0

  git clone https://github.com/saurabhmishra/ambari-vagrant.git
  cd ambari-vagrant
  cd centos6.5
  ./up.sh
